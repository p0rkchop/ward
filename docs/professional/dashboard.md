# Professional Dashboard

The Professional Dashboard gives you a quick overview of your shift schedule and client bookings.

---

## Stats Cards

The dashboard displays key metrics:

| Metric | Description | Click Action |
|:-------|:------------|:-------------|
| **Available Timeslots** | Number of open timeslots across your shifts | Navigate to shifts |
| **Upcoming Appointments** | Client bookings in the next 48 hours | Navigate to bookings |

---

## Dashboard Components

### Recent Bookings

Shows the latest client appointments assigned to your shifts, including:
- Client name
- Appointment date and time
- Resource information

### Upcoming Shifts

Displays your shifts for today and tomorrow:
- Shift time range
- Resource name and location
- Number of bookings on each shift

### Shifts Table

A quick-view table of your active shifts with booking counts and status.

---

## Using the Dashboard

The dashboard is your starting point each day. Use it to:

- **See your schedule at a glance** — upcoming shifts and appointments
- **Identify busy periods** — check which shifts have the most bookings
- **Navigate quickly** — click stat cards to jump to detailed views
