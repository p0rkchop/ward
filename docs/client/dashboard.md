# Client Dashboard

The Client Dashboard provides a quick summary of your appointment activity.

---

## Stats Cards

The dashboard displays key metrics:

| Metric | Description | Click Action |
|:-------|:------------|:-------------|
| **Booked Appointments** | Total number of your confirmed appointments | Navigate to appointments |
| **Upcoming Appointments** | Appointments in the next 24–48 hours | Navigate to appointments |

---

## Next Appointment

If you have an upcoming appointment, the dashboard highlights it with:

- **Date and time** of the appointment
- **Professional name** assigned to you
- **Resource** name and location
- Quick action to view details or cancel

---

## Quick Actions

From the dashboard, you can:

- **Book a new appointment** — Jump directly to the booking page
- **View all appointments** — See your complete booking history
- **Cancel an upcoming booking** — Manage existing appointments
