# Analytics

The Analytics page provides system-wide statistics and trends to help you understand how Ward is being used.

---

## Accessing Analytics

Navigate to **Analytics** from the admin navigation bar. This page is **admin-only** — professionals and clients cannot access it.

---

## Available Metrics

The analytics page displays aggregate data about your Ward system, including:

- User registration trends
- Booking volume over time
- Resource utilization rates
- Event participation metrics

{: .note }
Analytics reflect real-time data from your system. Use these insights to optimize resource allocation, identify popular time slots, and plan future events.
