# Admin Dashboard

The Admin Dashboard provides a high-level overview of your Ward system with key metrics displayed as interactive stat cards.

---

## Stats Cards

The dashboard displays four primary metrics:

| Metric | Description | Click Action |
|:-------|:------------|:-------------|
| **Total Active Users** | Count of all non-deleted users in the system | Navigate to Users page |
| **Active Resources** | Count of all active (non-deleted) resources | Navigate to Resources page |
| **Upcoming Events** | Count of events with future end dates | Navigate to Events page |
| **Total Bookings** | Count of all confirmed bookings | Navigate to Events page |

Each card is clickable and navigates to the relevant management page for deeper detail.

---

## Using the Dashboard

The dashboard is your starting point after logging in. Use it to:

- **Quickly assess system health** — see if resources, events, and users are set up
- **Spot issues** — zero bookings or missing resources indicate setup may be needed
- **Navigate efficiently** — click any stat card to jump directly to that section

{: .note }
Stats reflect real-time data. Soft-deleted records (users, resources, events) are excluded from counts.
